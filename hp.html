HP

  • HP = ((Class Base HP*CON Modifier)*Buffs*Weapon Bonus)+Boost HP+Armor Bonus+M.Def. Bonus
  • The final HP is rounded down.

Class Base HP

Note - if you're a deleveled class, such as a level 30 Treasure Hunter, you would use the same HP chart as your pre-40 class - Rogue. All 76+ classes use the same HP charts as their respective 40+ classes.
