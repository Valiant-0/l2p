Shield Block Rate

  • Shield Block Rate = Base Shield Block Rate*DEX Modifier*Shield Mastery*Buffs*Set Bonus
  • If the attacker is using a bow, multiply the Shield Block Rate by 3.
  • The following skills "have a chance" to ignore shield block rate - Spinning Slasher, Thunder Storm, Punch of Doom, Hammer Crush, Triple Sonic Slash, Burning Fist, Soul Breaker, Hurricane Assault, Crush of Doom, Shock Blast, and Earthquake
